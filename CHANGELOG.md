# Changelog

## Unreleased

### Изменено
- Сборка перенесена на WXT с отдельными пакетами Chrome MV3 и Firefox MV3.
- Добавлены команды разработки, сборки и упаковки для обоих браузеров.
- Firefox получает собственный background script, extension ID и декларацию обрабатываемых данных.
- Кэш ответов ограничен 100 записями и сроком хранения 7 дней.
- Удалены старый ручной сборщик, отслеживаемый `dist`, дублирующий `full_project.txt` и неиспользуемая функция вставки текста.

### Проверено
- Добавлена автоматическая проверка Chrome/Firefox-манифестов.
- Все 6 Playwright-сценариев проходят на новой Chrome-сборке.

## [2.7.1] - 2026-07-07

### 🚀 Added (Добавлено)
- **E2E Автоматизация:** Внедрен полноценный набор E2E-тестов на базе Playwright для проверки UI и логики расширения.
- **Мокинг API:** Настроен перехват сетевых запросов (Mock) к Mistral API. Тесты больше не расходуют реальные токены и работают мгновенно.
- **Service Worker Intercept:** Реализован обходной путь для инъекции тестовых API-ключей и эмуляции кликов контекстного меню через фоновый скрипт (`background.ts`).

### ✅ Tested Features (Покрыто тестами)
- Защита системы (обработка отсутствующего API-ключа).
- Проверка орфографии (Alt+R).
- Переписывание стиля текста (Alt+Y).
- Добавление эмодзи (Alt+T).
- Эмуляция контекстного меню (Перевод текста).
- Негативные сценарии (корректная обработка HTTP 500 Internal Server Error от Mistral API).

### ⚠️ Known Issues (Известные проблемы)
- **OCR Тест (Alt+S):** Временно переведен в статус `.skip`. Нативная система фокуса браузера и системные окна (например, Google Translate) конфликтуют с эмуляцией мыши при отрисовке области "Ножниц" в автоматическом режиме.


## [2.7.0] - 2026-07-06

### ✨ Добавлено (Added)
- **Умный OCR («Ножницы»)**: добавлена функция захвата экрана и распознавания текста с картинок.
- **Интеграция с Pixtral-12b**: для режима OCR внедрена новейшая визуальная мультимодальная модель от Mistral.
- **Глобальный доступ**: вызов "Ножниц" теперь доступен через глобальный хоткей `Alt+S` и через системное контекстное меню (ПКМ) на любой странице.
- **Авто-копирование**: распознанный с картинки текст теперь мгновенно и автоматически копируется в буфер обмена.

### 🛠 Исправлено (Fixed)
- Исправлена проблема невидимости UI-панели при вызове окна через кастомные события (добавлен `position: fixed` для режима OCR).
- Отключено кэширование для визуальных запросов во избежание конфликтов состояний.
- Исправлено парсинг-преобразование Markdown-ссылок, вызывавшее ошибку `net::ERR_FILE_NOT_FOUND` при обращении к Mistral API.

## [2.6.0] - 2026-07-06

### ✨ Добавлено (Added)
- **i18n (Локализация)**: добавлена поддержка мультиязычности через `_locales`. Расширение готово к переводу на другие языки (Chrome Web Store Ready).
- **Умный UX для API-ключа**: если ключ не настроен, расширение теперь показывает красивое предупреждение и автоматически (через 3 секунды) перенаправляет пользователя на страницу настроек.
- **Контекст страницы**: нейросеть теперь знает заголовок и URL вкладки, на которой работает пользователь, для более точных и контекстных ответов.

### 🛠 Исправлено (Fixed)
- Исправлена критическая ошибка `Failed to fetch`, возникавшая из-за некорректного URL в Service Worker.
- Устранена ошибка `Uncaught (in promise) Error` при вызове горячих клавиш.
- Исправлено пропадание UI-панели (архитектура контент-скриптов возвращена к монолитному `content.ts` для максимальной стабильности в DOM).
- Продвинутый парсер: ИИ-теги `<b>` и `<strong>` теперь перехватываются и безопасно превращаются в зеленую подсветку `<mark>`, предотвращая XSS и вывод сырого HTML.
- Обновлены системные промпты (инструкции) для более строгого соблюдения ИИ формата выделения текста двойными звездочками.

## [2.5.1] - 2026-07-04

### ✨ Добавлено (Added)
- **История запросов**: создана отдельная страница истории (сохраняет до 50 последних успешных результатов ИИ).
- **Кнопка "Глазок"**: добавлена возможность показывать/скрывать API-ключ в настройках для удобства копирования.
- **Продвинутый Markdown парсер**: нейросеть теперь корректно форматирует списки (`- ` и `1. `) и абзацы, переводя их в чистый HTML (`<ul>`, `<li>`, `<br>`).

### 🛡 Безопасность и Валидация (Security)
- **Тест-драйв API-ключа**: добавлена предварительная валидация Mistral API ключа (через тестовый запрос) перед его сохранением в `chrome.storage`. 

### 🛠 Исправлено (Fixed)
- Исправлены ошибки строгой типизации TypeScript (`unknown` -> `any[]`) при работе с массивами данных из браузерного хранилища.

## [2.5] - 2026-07-04 (Текущая дата)

### ✨ Добавлено (Added)
- **Полная поддержка Google Docs**: реализован обход ограничений закрытого Canvas через чтение системного буфера обмена.
- **Интеграция в системное контекстное меню**: расширение теперь доступно по нажатию правой кнопки мыши (ПКМ).
- **Горячие клавиши (Hotkeys)**: добавлены официальные системные команды Chrome (Alt+R, Alt+Y, Alt+T) для быстрого вызова ИИ.
- **Сохранение Rich Text форматирования**: при изменении стиля или исправлении ошибок нейросеть сохраняет оригинальные HTML-теги (жирный шрифт, курсив, ссылки).
- **Инфраструктура тестирования**: внедрен фреймворк Jest для автоматизированного unit-тестирования модулей парсинга.

### 🛠 Исправлено (Fixed)
- Исправлен баг "залипания" всплывающего окна (Drag & Drop) при отпускании левой кнопки мыши.
- Устранена ошибка смешивания контекста: нейросеть больше не возвращает лишний текст вокруг выделенного слова.
- Исправлено поведение всплывающей панели в сложных `input` и `textarea` полях.

### ♻️ Изменено (Changed)
- Улучшена архитектура: логика парсинга ответов вынесена в отдельную утилиту `src/utils/textParser.ts`.
- Обновлен системный промпт модели `mistral-large-latest` для более строгого следования формату без лишних приветствий.
