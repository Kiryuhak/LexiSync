Укажите ваш персональный API ключ от Mistral AI. Он будет надежно сохранен в локальном хранилище браузера.

Где взять API ключ?

  1. Перейдите на console.mistral.ai и зарегистрируйтесь.
  2. В меню слева найдите раздел API Keys.
  3. Нажмите Create new key, скопируйте его и вставьте в поле выше.
